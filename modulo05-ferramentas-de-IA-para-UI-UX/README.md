# 🚀 Ferramentas de IA para UX & UI

Este repositório centraliza os projetos, artefatos e automações desenvolvidos durante a disciplina de **IA Aplicada a UX e UI**. O foco central é a aplicação da Inteligência Artificial como uma camada de **Engenharia de Software**, otimizando o Discovery, a Prototipação e o Desenvolvimento Front-End.

**Professor:** [Alvaro Camillo Neto](https://github.com/alvarocamilloneto)

---

## 📂 Estrutura do Repositório

O curso é organizado em módulos que acompanham a evolução de um produto digital, desde o refinamento de requisitos até a integração de lógica de IA no cliente e servidor.

```bash
.
├── modulo-01-discovery-refinement/ # Refinamento de Requisitos e Data Discovery
├── modulo-02-prototyping-ui/     # Figma to Code e Firebase Studio
├── modulo-03-agents-cli/         # Gemini CLI e Agentes de Refatoração
├── modulo-04-automation-mcp/     # Testes E2E e Automação com MCP
└── modulo-05-ai-integration/     # Firebase AI Logic e Integração de APIs
```

Engine: Google Gemini (1.5 Pro / Flash)

Documentation: Mermaid.js para fluxos lógicos

Standard: JSON Prompts (Padrão Erick Wendel)

---

## 🎯 Objetivo dos Projetos

Diferente de uma abordagem de design tradicional, aqui tratamos a IA como uma ferramenta de redução de variabilidade e estresse de lógica.

Módulo 1: Refinar requisitos ambíguos e transformar dados de pesquisa desestruturados em backlog priorizado via Structured Prompts.

Módulo 2: Automatizar o handoff de design para código utilizando aceleradores baseados em IA.

Módulo 3: Utilizar CLI e Agentes para scaffolding e refatoração de código legado.

Módulo 4: Expor o contexto da aplicação via MCP para que agentes autônomos realizem testes e depuração.

Módulo 5: Implementar lógica de IA no backend sem gerenciar infraestrutura complexa.
